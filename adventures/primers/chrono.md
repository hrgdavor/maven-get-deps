# 🌀 Tome of Temporal Shifting
*A Primer on Atomic Transitions*

To complete the Chrono-Shifter's Leap, you must master the art of symbolic shifting.

## 🔗 The Symbolic Link (Symlink)
A symlink is a pointer. Instead of pointing to data, it points to another path.
Example: `/opt/app/current` -> `/opt/app/versions/v1.10`.

## ⚡ The Atomic Swap
Swapping a symlink is "atomic." It happens instantaneously. The system doesn't see a partially changed path; it sees the old path one microsecond and the new path the next.

## 📅 Versioned Sanctuaries
Never overwrite an existing deployment. Always deploy to a unique folder (named by version or timestamp). This allows for instant **Rollback** by simply pointing the symlink back to the previous sanctuary.
