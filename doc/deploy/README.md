# Deploy your app

You should learn to deploy your app the old fashioned way on a VPS, or bare metal server without just offloading your code to devops or whatever currently fancy setup is all the rage (Docker, Kubernetes,...). 

Invest some time into this and it will give you a better understanding of those fancy things if you end-up using them anyway.

> Start with [running your app locally](local.md) without IDE, to experience initial basic steps to start an application before continuing in this document to learn about deploying to a server.

After you have mastered running your app locally, there are more things you should learn to setup your app on a server and maintain it. Paradoxically you will look at this document thinking how it is complicated, and in the same breath think you will do it more easily with docker and Kubernetes.
